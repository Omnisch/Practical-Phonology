# 大话音系：实用语音学入门

- 本栏目为 Omnisch 个人编纂的实用语音学手册，旨在传授「零基础[^1]也能理解，且能应用于日常生活的语音学、语言学知识」。

- **出于「实用」之目的，本栏目将以问答的形式科普、讲解语音学或语言学现象，而不会深入讨论这些现象的背景及成因。**
